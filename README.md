## Repositório

### Clone

```
git clone --bare https://github.com/kaikcreator/rxjs-school rxjs-nivel-pro/.git
cd .\rxjs-nivel-pro\
```

### Reset Repositório

```
git config --unset core.bare
git reset --hard
```

### Instalar Dependências

```
npm i
```

```
cd angular-rxjs
git config --unset core.bare
git reset --hard
git checkout dev/01-rxjs-in-angular
npm i
npm i -g json-server
json-server --watch db.json
npm start
```
